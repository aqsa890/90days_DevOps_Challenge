## 📖 About This Repository

Welcome to my **#90DaysOfDevOps** journey!

This repository documents my daily progress as I learn **Linux, Networking, AWS, Docker, Kubernetes, CI/CD, Shell Scripting, Python, Terraform, Jenkins**, and other DevOps tools through hands-on practice and real-world projects.

The goal is not just to complete the challenge, but to build a strong foundation in DevOps by staying consistent and sharing my learning publicly.

> **"Consistency beats intensity. Small progress every day leads to big results." 🚀**
